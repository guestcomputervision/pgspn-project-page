for Sparsity-Robust Underwater Depth Completion

Abstract

Underwater robots require dense metric depth for mapping, navigation, and manipulation, yet practical sources such as Structure-from-Motion (SfM) and field-of-view-limited sensors provide only sparse, spatially biased measurements with frame-varying density. Existing depth completion methods assume uniformly distributed inputs and degrade severely under such irregular sparsity. Underwater scenes further aggravate this problem because expansive open-water regions lack ground-truth supervision, causing estimated depths to erroneously extend beyond object boundaries into empty space—a phenomenon we term depth bleeding. We propose PGSPN, a physics-guided depth completion framework that exploits underwater optical attenuation to construct a sparsity-robust metric depth prior from per-channel transmission maps. Our Physics-Guided Propagation Suppression (PGPS) module explicitly prevents depth bleeding by attenuating propagation across transmission-derived depth discrepancies. We additionally introduce the Depth Bleeding Index (DBI), a new metric for quantifying boundary preservation in open-water regions. Evaluated on real underwater datasets with sequence-level splits, cross-dataset generalization, and real sensor imagery, PGSPN reduces MAE by up to 47% over the best baseline while achieving the lowest DBI across all conditions.

Key Insights

Irregular Sparsity in Underwater Environments

Unlike terrestrial LiDAR which provides uniformly distributed depth, underwater sources (SfM, sonar) produce spatially biased, frame-varying sparse measurements. Large open-water regions lack any depth supervision, causing severe depth bleeding artifacts.

Physics-Guided Metric Prior

We exploit the Beer–Lambert law: underwater light attenuation embeds depth-dependent photometric cues in RGB images. By estimating per-channel transmission maps and rescaling to metric scale, we obtain a dense, boundary-preserving depth prior independent of sparse input density.

Depth Bleeding Suppression

Our PGPS module leverages the transmission-based prior to modulate spatial propagation weights—fully preserving propagation within homogeneous regions while exponentially suppressing it across depth discontinuities at open-water boundaries.
